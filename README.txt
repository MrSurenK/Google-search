Google Search Project

What is this project about?
This project is my very 1st project uploaded on Github to practice and apply my CSS and HTML knowledge.

Objective:
1. Clone Google Search home page using only HTML and CSS 

Resources used:
1.https://www.youtube.com/watch?v=fYq5PXgSsbE - Understanding CSS Flexbox
2.https://www.youtube.com/watch?v=ZY1MYwUDZsY&t=509s - Google HTML structure 
3.https://www.youtube.com/watch?v=5EJWYUwOe3E - Google CSS and some HTML structure guide

**The code was not exclusively copied from any one resource. It was only used as a guide to help me along in organising and formatting my code**

Features:
- Mobile friendly
A)Page content would resize and transition smoothly to fit smaller screen sizes without any compromise on formatting. 
B)Hover effects on search bar and clickable icons as per the original Google site. 
C)Footer will reorganise itself when browser size is shrunk as per original Google site.
